Плагин для Wordpress для добавления шорткодов галареи или сладера на страницу новости.

== Description ==

Плагин создает шорткод галареи(слайдера) который можно добавить в редаторе гутенберг на страницу с новостями.

= Features =
* Использует бибилиотеку OwlCarousel2
